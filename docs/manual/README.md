# Packset Operator's Manual

How Packset behaves and how to use it, end to end. This manual assumes you are a
licensed amateur operator who has used FT8 before (WSJT-X or similar) and is
new to Packset. It documents the current **`unstable`** channel behavior; the
stable channel may lag behind what is described here.

## Chapters

1. [Getting started](getting-started.md) — install, hook up the rig,
   configure, and see your first decodes.
2. [Operating](operating.md) — the Operate and Spectrum tabs: status bar,
   decode list, calling, answering, and QSO automation.
3. [Settings reference](settings.md) — every setting: default, behavior,
   interactions.
4. [Logging and ADIF](logging.md) — what gets logged, the Log tab, exports,
   POTA activation files, backups, and import.
5. [Troubleshooting](troubleshooting.md) — symptom-first fixes.
