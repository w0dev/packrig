The parser only needs to handle the specific fixed schema in RECORDING-FORMAT.md — quoting rules: strings are double-quoted with no embedded quotes, numbers are bare integers or decimals, no nested objects in payload (string-to-string only), no escape sequences other than basic ASCII.     Create app/src/test/java/net/ft8vc/app/foundations/golden/GoldenTraceReplay.kt. Package: `net.ft8vc.app.foundations.golden`. Imports: the Phase 0 fakes (FakeRigBackend, FakeUsbAudioCapture, Ft8DecoderFake), real domain types (QsoMachine, QsoMessages, QsoState, QsoDecode, AnswerPolicy from net.ft8vc.core), Ft8DecodeResult.

    Class: `class GoldenTraceReplay(val events: List<TraceEvent>, val machine: QsoMachine, val rig: FakeRigBackend = FakeRigBackend(), val capture: FakeUsbAudioCapture = FakeUsbAudioCapture(), val decoder: Ft8DecoderFake = Ft8DecoderFake())`. Public `fun run(): ReplayResult`.

    `run()` algorithm (event-driven, virtual time — no real sleeps):
    1. Iterate events in order.
    2. On TRACE_START: extract myCall/myGrid from payload (informational; machine already has them).
    3. On DECODE_BATCH: build `List<QsoDecode>` by mapping TraceDecode → QsoDecode(message, snr); pass to `machine.onDecodes(decodes)`. Also queue corresponding Ft8DecodeResult list into decoder via decoder.queueDecodeResults so subsequent decoder.decode() returns it (parity with Phase 3+ controllers that drive through decoder). Increment observedDecodeBatches.
    4. On UI_ACTION: read payload["action"]; for action == "answer_cq", call `machine.answerCq(payload["dx_call"]!!, payload["dx_grid"])`; for action == "start_cq", call `machine.startCq()`.
    5. On TX_SLOT: if `machine.txMessage() != null`, call rig.keyPtt(); then call machine.markTransmitted(); then call rig.releasePtt(). The keyPtt/releasePtt sequence drives the FakeRigBackend's pttEdges history.
    6. On TRACE_END: stop iteration.

    `data class ReplayResult(val finalState: QsoState, val pttKeyEdges: Int, val pttReleaseEdges: Int, val observedDecodeBatches: Int, val pttEdges: List<net.ft8vc.rig.fakes.PttEdge>, val decodeInvocations: List<net.ft8vc.ft8native.fakes.DecodeInvocation>)`. Populated from rig.pttEdgesSnapshot() and decoder.decodeInvocationsSnapshot() at end of run(); pttKeyEdges/pttReleaseEdges are derived counters.
